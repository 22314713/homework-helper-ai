# Homework Helper AI Agent

A basic AI agent built with Python that helps with homework by researching topics and providing summaries of the best parts.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Create a `.env` file in the project root and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
   Get a free API key from [Google AI Studio](https://makersuite.google.com/app/apikey).

3. Run the agent:
   ```
   python main.py
   ```

## Usage

Enter a topic when prompted, and the agent will generate a summary.
